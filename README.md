# leka-med-sklearn

Minimal working example of sklearn with some of the preprocessing features one can use (on demo data)


#### Dimenasionality reduction techniques
* tSNE
* PCA
* kPCA

#### ML
 
* Regression
  * sci-kit RFregressor
  * XGBoost regressor
* Classification
  * sci-kit RFClassifier
  * XGBoost classifier
